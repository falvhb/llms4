# The missing llms.txt directory

Feel free to add missing llms.txt via PR

Format: `(www.)domain.tld.md`

## Hints:

- As names, use the top domains including www subdomain if used
- This is intended for locally running LLMs, so keep it simple
- Pure site-directory style llms.txt files are ignered, just let your agent read the sitemap
- All PRs are hand-reviewed, there are not so many useful websites out there
